    /players/ - возвращает массив игроков

    /players/ (POST) - принимает json с одним игроком на добавление, возвращает его же

    /players/{id} - возвращает одного игрока с указанным id (/players/2)

    -------------------------

    /sessions/ - массив сессий

    /sessions/ (POST) - принимает json с одной сессией на добавление

    /sessions/{id} - одная сассия с указанным id (/sessions/1)

    /sessions/{sessionId}/add/{playerId} (PUT) - добавляет игрока в сессию по указанным id (полное говно, скоро переделаю)

    -------------------------

    /action/rolldice/ (PUT) - принимает json указанного выше типа, возвращает результат броска костей
